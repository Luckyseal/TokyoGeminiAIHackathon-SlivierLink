# SilverLink Ambient Agent

Ambient Family Intelligence for aging societies. Built for the Gemini AI Hackathon Tokyo 2026.

## Structure
- `app/`: Cloud Run-ready React + TypeScript app with server-side Vertex AI Gemini 3.5 Flash, Google Cloud Text-to-Speech, wellbeing summaries, and care memory.
- `working/`: Local-only planning materials and runbooks. This directory is intentionally excluded from GitHub.

For detailed setup, Google Cloud API integration info, and demo script, please see `app/README.md`.
